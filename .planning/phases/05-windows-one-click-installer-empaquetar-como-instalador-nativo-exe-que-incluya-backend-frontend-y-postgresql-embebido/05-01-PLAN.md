  </read_first>
  <action>
    Create `electron/main.js` — the core orchestrator (per D-01, D-03, D-07):

    Constants:
    - BACKEND_PORT = 8765
    - BACKEND_HOST = '127.0.0.1'
    - PG_PORT = 5432 (with fallback to 5433 if 5432 is occupied — check with net.createServer)
    - HEALTH_URL = `http://${BACKEND_HOST}:${BACKEND_PORT}/health`

    Helper `getResourcesPath()`:
    - Returns `process.resourcesPath` if `app.isPackaged`, else `path.join(__dirname, 'resources')`

    Helper `runCommand(cmd, args)`:
    - Returns a Promise wrapping `child_process.execFile(cmd, args)` with stdout/stderr logging via electron-log

    Startup sequence in `app.whenReady()`:
    1. Check for port availability (5432, fallback 5433) — use `net.createServer().listen(port)` test
    2. Call `startPostgres()`: Check for `PG_VERSION` file in `path.join(app.getPath('userData'), 'postgres-data')`; if missing, run `initdb.exe -D {pgData} -U postgres -A trust -E UTF8`; then run `pg_ctl.exe start -D {pgData} -l {pgLog} -o "-F -p {PG_PORT} -h 127.0.0.1"`
    3. Create database if not exists: run `createdb.exe -U postgres -h 127.0.0.1 -p {PG_PORT} dianfacturador` (ignore error if already exists)
    4. Call `startBackend()`: spawn backend.exe as child process with env vars: DATABASE_URL=`postgresql+asyncpg://postgres@127.0.0.1:{PG_PORT}/dianfacturador`, SECRET_KEY from secrets.js, FERNET_KEY from secrets.js, CERT_STORAGE_PATH=`path.join(userData, 'certs')`, BACKEND_PORT=8765, BACKEND_HOST=127.0.0.1, WEASYPRINT_DLL_DIRECTORIES=`path.join(resourcesPath, 'backend', 'gtk-dlls')`
    5. Call `waitForBackend(30000)`: poll HEALTH_URL every 500ms, max 30s
    6. If healthy: create BrowserWindow (1200x800, preload.js, nodeIntegration:false, contextIsolation:true), load `frontend/dist/index.html`
    7. If NOT healthy: show dialog.showErrorBox with backend stderr, then app.quit()
    8. Call `createTray(mainWindow)` from tray.js
    9. Set up auto-launch (per D-03): `const AutoLaunch = require('auto-launch'); new AutoLaunch({ name: 'DIAN Facturador', isHidden: true })`
    10. Call `setupAutoUpdater(mainWindow)` from updater.js (placeholder — implemented in Plan 04)

    Window close behavior (per D-03): `mainWindow.on('close', e => { e.preventDefault(); mainWindow.hide(); })` — closing the window hides to tray. Actual quit only from tray menu or Cmd+Q.

    Graceful shutdown in `app.on('before-quit')`:
    1. Kill backendProcess (SIGTERM on Unix, taskkill on Windows)
    2. Run `pg_ctl stop -D {pgData} -m fast`
    3. Call `app.exit(0)`

    Orphan cleanup on startup: Before starting PG, check if port PG_PORT is in use. If so, attempt `pg_ctl stop` on the existing data dir. This handles the crash-recovery case (Pitfall 9).

    Create `electron/preload.js`:
    - Use `contextBridge.exposeInMainWorld('electronAPI', { ... })` to expose:
      - `getBackendUrl()` returns `http://127.0.0.1:8765`
      - `onUpdateAvailable(callback)` — ipcRenderer.on('update-available', callback)
      - `onUpdateDownloaded(callback)` — ipcRenderer.on('update-downloaded', callback)
      - `restartToUpdate()` — ipcRenderer.send('restart-to-update')

    Create `electron/tray.js` (per D-03):
    - Export `createTray(mainWindow)` function
    - Icon: `path.join(__dirname, 'build', 'icon.ico')` (or a placeholder `nativeImage.createEmpty()` if icon not yet present)
    - Context menu with: "Abrir DIAN Facturador" (show+focus window), separator, "Salir" (app.quit())
    - Tooltip: "DIAN Facturador"
    - Double-click: show+focus window

    Create `electron/secrets.js`:
    - Export `getSecretKey(userData)` and `getFernetKey(userData)`
    - Each checks for `.secret_key` / `.fernet_key` file in userData
    - If file exists, read and return trimmed content
    - If not, generate: SECRET_KEY = `crypto.randomBytes(32).toString('hex')`, FERNET_KEY = `Buffer.from(crypto.randomBytes(32)).toString('base64url').slice(0,44)`
    - Write file with `fs.writeFileSync(path, value, { mode: 0o600 })`
    - Create userData directory with `fs.mkdirSync(userData, { recursive: true })` if needed
  </action>
