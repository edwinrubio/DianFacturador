    Create `backend/electron_entrypoint.py`:
    ```python
    import sys
    import os
    import subprocess

    # PyInstaller sets sys._MEIPASS for the extracted bundle location
    if hasattr(sys, '_MEIPASS'):
        os.chdir(sys._MEIPASS)

    # Run Alembic migrations before starting the server (per D-07)
    def run_migrations():
        """Run alembic upgrade head. Raises on failure."""
        result = subprocess.run(
            [sys.executable, '-m', 'alembic', 'upgrade', 'head'],
            capture_output=True, text=True,
            cwd=os.getcwd()
        )
        if result.returncode != 0:
            raise RuntimeError(f"Migration failed: {result.stderr}")

    if __name__ == '__main__':
        run_migrations()

        import uvicorn
        from app.main import app

        port = int(os.environ.get('BACKEND_PORT', '8765'))
        host = os.environ.get('BACKEND_HOST', '127.0.0.1')
        uvicorn.run(app, host=host, port=port, log_level='info')
    ```

    NOTE: The migration runner uses `subprocess.run` with `sys.executable -m alembic` because in a PyInstaller bundle, alembic's CLI entry point may not be directly available. If this approach fails during testing, the fallback is to use `alembic.config.main(['upgrade', 'head'])` programmatically.

    Create `backend/build_backend.bat` — a Windows batch script for building the PyInstaller bundle locally or in CI:

    ```bat
    @echo off
    REM Build the FastAPI backend as a standalone .exe using PyInstaller
    REM Must be run from a Windows machine with Python 3.12+ and MSYS2 GTK DLLs available

    echo [1/4] Installing Python dependencies...
    pip install -r requirements.txt
    pip install pyinstaller==6.19.0 pyinstaller-hooks-contrib

    echo [2/4] Running PyInstaller...
    pyinstaller backend.spec --distpath dist --clean

    echo [3/4] Copying GTK DLLs for WeasyPrint...
    if not exist "dist\backend\gtk-dlls" mkdir dist\backend\gtk-dlls
    REM Copy from MSYS2 — adjust path if MSYS2 is installed elsewhere
    set MSYS2_BIN=C:\msys64\mingw64\bin
    for %%d in (libpango-1.0-0.dll libpangocairo-1.0-0.dll libpangoft2-1.0-0.dll libcairo-2.dll libcairo-gobject-2.dll libgdk_pixbuf-2.0-0.dll libglib-2.0-0.dll libgobject-2.0-0.dll libgio-2.0-0.dll libharfbuzz-0.dll libfontconfig-1.dll libfreetype-6.dll zlib1.dll libffi-8.dll libbz2-1.dll libintl-8.dll libpcre2-8-0.dll) do (
        if exist "%MSYS2_BIN%\%%d" (
            copy "%MSYS2_BIN%\%%d" dist\backend\gtk-dlls\
        ) else (
            echo WARNING: %%d not found in %MSYS2_BIN%
        )
    )

    echo [4/4] Build complete!
    echo Output: dist\backend\backend.exe
    dir dist\backend\backend.exe
    ```

    This script is used both locally by developers on Windows and by GitHub Actions CI.
