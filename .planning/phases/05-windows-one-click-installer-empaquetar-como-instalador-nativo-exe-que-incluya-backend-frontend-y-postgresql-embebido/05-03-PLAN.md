    Change the hardcoded `baseURL: "/api"` to read from the environment variable:
    ```typescript
    const api = axios.create({
      baseURL: import.meta.env.VITE_API_BASE_URL || "/api",
      headers: { "Content-Type": "application/json" },
    });
    ```
    This preserves the Docker behavior (VITE_API_BASE_URL is not set -> falls back to "/api" with nginx proxy) while allowing the Electron build to use the direct URL.

    Add a `build:electron` script to `frontend/package.json` scripts:
    ```json
    "build:electron": "vite build --mode electron"
    ```
    Vite's `--mode electron` flag makes it load `.env.electron` automatically (Vite convention: `.env.[mode]`).

    Do NOT modify `vite.config.ts` — the proxy config is dev-only and the build config does not need changes.
  </action>
