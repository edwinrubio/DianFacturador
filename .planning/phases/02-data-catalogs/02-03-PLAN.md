**Model (backend/app/models/product.py):**
```python
from decimal import Decimal
from sqlalchemy import Boolean, Numeric, String, Text
from sqlalchemy.orm import Mapped, mapped_column
from app.models.base import Base

class Product(Base):
    __tablename__ = "products"

    id: Mapped[int] = mapped_column(primary_key=True, autoincrement=True)

    # Identification
    internal_code: Mapped[str] = mapped_column(String(50), nullable=False, unique=True)
    description: Mapped[str] = mapped_column(Text, nullable=False)

    # Pricing — CRITICAL: Numeric, NEVER float
    unit_price: Mapped[Decimal] = mapped_column(Numeric(18, 2), nullable=False)

    # Tax — DIAN tax codes (Tabla 13.2.11)
    tax_code: Mapped[str] = mapped_column(String(5), nullable=False, default="01")
    # "01" = IVA (only option for Phase 2)
    tax_rate: Mapped[Decimal] = mapped_column(Numeric(5, 2), nullable=False, default=Decimal("19.00"))
    # Valid IVA rates: 0.00, 5.00, 19.00

    # Unit of measure — UNECE Rec. 20 code
    unit_measure_code: Mapped[str] = mapped_column(String(5), nullable=False, default="94")
    # "94" = Unidad (default)

    # Optional
    standard_code: Mapped[str | None] = mapped_column(String(50), nullable=True)
    # UNSPSC code — optional per CONTEXT.md decision

    is_active: Mapped[bool] = mapped_column(Boolean, default=True)
```

**Router (backend/app/routers/products.py):**
`APIRouter(prefix="/api/products", tags=["products"])`

Same CRUD pattern as resolutions.py:
1. `GET /api/products` → list all active products, ordered by description. Support optional `q: str` for search (ILIKE on description or internal_code).
2. `POST /api/products` → create. Return 201.
3. `GET /api/products/{id}` → get single. 404 if not found.
4. `PUT /api/products/{id}` → update.
5. `DELETE /api/products/{id}` → soft delete (set is_active=False). Return 204.

All endpoints require `Depends(get_db)` and `Depends(get_current_user)`.

**Migration:** `alembic revision --autogenerate -m "create products table"`.
