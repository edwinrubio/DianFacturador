**Service (backend/app/services/import_service.py):**

Core parsing function:
```python
def parse_file(content: bytes, content_type: str) -> list[dict]:
```
- For `text/csv`: decode with `utf-8-sig` first, fallback to `latin-1` on UnicodeDecodeError. Use `csv.DictReader`.
- For xlsx: use `openpyxl.load_workbook(filename=io.BytesIO(content), read_only=True)`. Read headers from row 1, data from row 2+. Normalize headers to lowercase stripped.
- Return list of dicts.

Client import function:
```python
async def process_client_file(content: bytes, content_type: str, db: AsyncSession) -> ImportResult:
```
- Parse file
- For each row (track row number starting at 2 for data rows):
  - Map CSV columns to model fields: tipo_documento→document_type, numero_documento→document_number, digito_verificacion→check_digit, razon_social→business_name, nombre_comercial→trade_name, email→email, telefono→phone, direccion→address_line, codigo_departamento→department_code, codigo_municipio→municipality_code, responsabilidad_fiscal→fiscal_responsibilities, regimen_tributario→tax_regime
  - Validate required fields present (document_type, document_number, business_name, email, address_line, department_code, municipality_code)
  - Validate document_type in allowed set
  - If document_type == "31": validate check_digit present and correct using `nit.calculate_check_digit()`
  - Zero-pad department_code to 2 digits, municipality_code to 5 digits (CRITICAL: prevents leading zero loss from Excel)
  - Check for duplicate document_number in DB
  - If all valid: create Client object, add to session
  - If invalid: append ImportRowError, increment skipped
- Commit all valid rows at once
- Return ImportResult

Product import function:
```python
async def process_product_file(content: bytes, content_type: str, db: AsyncSession) -> ImportResult:
```
- Same pattern. Map: codigo_interno→internal_code, descripcion→description, precio_unitario→unit_price (convert to Decimal), codigo_impuesto→tax_code, tarifa_impuesto→tax_rate (convert to Decimal), unidad_medida→unit_measure_code
- Validate: internal_code required, description required, unit_price > 0, tax_rate in {0, 5, 19}
- Check duplicate internal_code in DB
- Return ImportResult

In `backend/app/routers/clients.py`, add:
```python
from fastapi import UploadFile, File

@router.post("/import", response_model=ImportResult)
async def import_clients(
    file: UploadFile = File(...),
    db: AsyncSession = Depends(get_db),
    current_user: User = Depends(get_current_user),
):
    if file.content_type not in ["text/csv", "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet"]:
        raise HTTPException(400, "Formato no soportado. Use CSV o Excel (.xlsx)")
    content = await file.read()
    return await import_service.process_client_file(content, file.content_type, db)
```

IMPORTANT: Place the `/import` route BEFORE `/{id}` routes to avoid FastAPI treating "import" as an ID path parameter.
